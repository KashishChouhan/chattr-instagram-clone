# Overview
Chattr is a mini Instagram-style app allowing users to signup/login, follow users, create posts, like, comment, and see a personalized feed.

## Tech Stack
- Backend: Node.js, Express, MongoDB, Mongoose
- Frontend: React, Vite
- Authentication: JWT, bcrypt

## Features
- Signup & Login
- Follow/Unfollow
- Create Post (Image URL + Caption)
- Like & Comment
- Personalized feed from followed users
